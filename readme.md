
## Prerequisites

- Java 8 or higher
- Maven 3.6.0 or higher

## Setup

1. **Clone the repository:**

    ```sh
    git clone https://github.com/vikas735s/swapi-tests.git
    cd swapi-tests
    ```

2. **Build the project:**

    ```sh
    mvn clean install
    ```

## Running Tests

To run the tests, use the following Maven command:

```sh
mvn test
